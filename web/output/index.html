Scrapism

This is a work-in-progress guide to web scraping as an artistic and critical practice, created by Sam Lavigne.

Schedule: Every few days I'll upload some new resources and videos to this page. I'll also be doing occasional live demos either on Twitch or YoutTube. You can sign up to receive updates at: https://tinyletter.com/scrapism

Prerequisites: Please note that this guide is intended for those with some basic programming experience. You definitely don't have to be a pro (I'm not), but you should know what a variable is. To that end, the first two sections act as brief overviews/refreshers of important topics you'll need to be able to follow the rest of the guide.
